# Política de Privacidade — Checks LGPD

## Checks obrigatórios

### Existência e acessibilidade
- [ ] Política de privacidade existe e está publicada
- [ ] Link acessível no rodapé de todas as páginas
- [ ] Não está atrás de login ou paywall
- [ ] Redigida em português claro e acessível
- [ ] Data de última atualização visível no topo

### Identificação do controlador
- [ ] Nome/razão social do controlador identificado
- [ ] CNPJ informado
- [ ] Endereço postal completo
- [ ] Canal de contato para questões de privacidade (email ou formulário)
- [ ] DPO/Encarregado nomeado com contato público (Art. 41)

### Dados coletados e finalidades
- [ ] Categorias de dados pessoais listadas (dados cadastrais, de uso, pagamento, técnicos)
- [ ] Dados sensíveis identificados separadamente, se aplicável (Art. 11)
- [ ] Finalidade específica declarada para cada categoria de dados
- [ ] Finalidades são legítimas, específicas e explícitas (Art. 6, I)

### Base legal
- [ ] Base legal identificada para cada finalidade de tratamento (Art. 7)
- [ ] Se usa consentimento: informado que é livre, informado, específico e revogável
- [ ] Se usa legítimo interesse: menção ao teste de proporcionalidade (legitimidade, necessidade, balanceamento)
- [ ] Não usa bases legais genéricas ou vagas ("para melhorar sua experiência")

### Compartilhamento e transferência internacional
- [ ] Destinatários/operadores listados por nome ou categoria
- [ ] Finalidade do compartilhamento declarada
- [ ] Transferências internacionais declaradas com mecanismo de proteção:
  - Decisão de adequação (UE desde jan/2026 — Resolução 32/2026)
  - SCCs da ANPD (Resolução 19/2024) para outros países
  - Consentimento específico e destacado
- [ ] Não afirma "não transferimos dados" enquanto usa SaaS estrangeiro

### Retenção
- [ ] Prazos de retenção informados por categoria de dados
- [ ] Critérios de determinação do prazo, se prazo exato não é possível
- [ ] Menção às exceções de retenção (Art. 16): obrigação legal, pesquisa, transferência, uso exclusivo anonimizado

### Direitos do titular
- [ ] Lista completa dos direitos do Art. 18:
  - Confirmação e acesso
  - Correção
  - Anonimização, bloqueio ou eliminação
  - Portabilidade
  - Informação sobre compartilhamento
  - Informação sobre possibilidade de negar consentimento
  - Revogação do consentimento
  - Petição à ANPD
- [ ] Direito de revisão de decisões automatizadas mencionado (Art. 20)
- [ ] Prazo de resposta informado (15 dias para acesso detalhado)
- [ ] Canal para exercício dos direitos claramente indicado

### Cookies e rastreamento
- [ ] Menção a cookies e tecnologias similares (ou link para aviso de cookies separado)
- [ ] Modelo opt-in declarado para cookies não essenciais

### Incidentes de segurança
- [ ] Menção ao compromisso de notificação em caso de incidente
- [ ] Referência ao prazo de 3 dias úteis (Resolução 15/2024)

### Crianças e adolescentes
- [ ] Se aplicável: menção ao tratamento de dados de menores (Art. 14)
- [ ] Consentimento parental/do responsável mencionado
- [ ] Tratamento no "melhor interesse" da criança declarado

### Decisões automatizadas e IA
- [ ] Se aplicável: divulgação do uso de profiling ou decisões automatizadas
- [ ] Direito de solicitar revisão por pessoa natural (Art. 20)
- [ ] Critérios e procedimentos utilizados descritos (quando solicitado)
